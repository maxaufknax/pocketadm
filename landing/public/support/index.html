Getting started

PocketADM has two parts: the app (this iOS app) and the server software you run on your own Linux box. Install the server with one command, then add it in the app:

  • On any Linux server, run the one-line installer from pocketadm.com. It prints an admin password.
  • In the app, tap Add a server, enter the address (e.g. http://your-server:8090) and that password.
  • Or, on a server you're already signed into, show a pairing QR and scan it from the app.

Try it without a server

Want to look around first? Add the demo server demo.pocketadm.com with the username and password demo — it's a read-only sandbox.

Common questions

Do I need a Mac or a subscription? No. PocketADM is free; you run the server on your own hardware and bring your own AI key (or a local model) if you want the assistant.

Can't the app reach my server? Make sure the server is running and reachable from your phone's network. For servers at home, connect over the same Wi-Fi or a VPN.
