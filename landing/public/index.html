Everything, from your phone

One app to run your whole server

No more SSH-ing from a laptop to fix a container at 11pm. Helmsman puts the whole box (metrics, logs, updates, terminal and an AI engineer) one tap away.

✦

Vibe Code: an AI that works on your server

Chat with an agent that runs commands, reads & writes files, and fixes things directly on your box. It streams live, keeps working when you close the app, and asks before anything destructive, or flip on Auto.

📊

Live dashboard

Real-time CPU, RAM, disk, uptime and internet latency with tap-through history graphs. Every container grouped by stack: start, stop, restart, logs, one-tap port links.

❯_

A real terminal

A full xterm terminal over WebSocket, with a mobile key bar and one-tap docker exec into any container. Open any command the agent ran to watch it yourself.

◲

One-tap App Store

Deploy popular self-hosted apps as clean compose projects with friendly config forms, and it detects apps you already run. Build something new and Helmsman spots the new service and helps you finish setup.

⟳

Smart updates & rollback

Detects newer images by registry digest (no pull needed) with AI release-note explanations. Snapshots the running image before every update so you can roll back in one tap.

🧠

Local AI, built in

Run models on your own hardware via Ollama: Helmsman detects or installs it, recommends models that fit your RAM, and wires them straight into the chat. Private, free, offline.

🛰

Background sentinels

Little AI watchmen check your server on a schedule (security, updates, health) and push findings to your phone via ntfy, each with a one-tap “discuss & fix”.

🖧

Multi-server & handoff

Manage several servers from one app. Pair a new device by QR, and hand a live agent chat from your phone to your laptop mid-task, and it keeps streaming.

🔒

Secure by design

Password + optional TOTP 2FA, token revocation, an append-only audit log of every action (including the AI’s), and integration tokens that are never shown to the model.

Install with one line

SSH into any Linux server. The installer sets up Docker (if needed) and Helmsman, then prints your admin password. Open it on your phone, add to home screen, done.

$ curl -fsSL https://raw.githubusercontent.com/maxaufknax/helmsman/main/install.sh | bash
1Run the installerDocker + Helmsman on port 8090, admin password printed to the console.
2Open on your phoneVisit http://your-server:8090 and add it to your home screen as a PWA.
3Add an AI key (optional)Bring an Anthropic / OpenRouter / OpenAI key, or run a local model, and start vibing.
